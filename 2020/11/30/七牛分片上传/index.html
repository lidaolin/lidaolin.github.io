今天说下七牛云的分片上传

项目中我们自己项目如果遇到大图片视频，等需要上传的项目会发现如果本地上传到自己服务器。
第一会发现上传的视频过大有可能会出现响应终止的情况，第二会发现自己服务器上传了太多的媒体素材不好管理，第三图片还得自己手动压缩和视频截取封面图反正麻烦的一批
七牛云的分片上传其实对前端的友好程度也不是多高，但是他便宜啊。接下来就看下七牛分片上传

七牛上传有点坑爹他没有自己的上传代码，他使用的是Plupload上传组件就是说用七牛上传最好使用Plupload进行数据流上传
七牛云上传使用方式有两种一种是用<script>引入一个是使用vue中npm安装js包进行

七牛云上传进行script引入模式

无需npm install安装

引入后就可以在vue文件中使用了这种使用方式比较简单点
参考vue+七牛云上传视频文件

另外种方法就比较麻烦先下载
npm install plupload -s
七牛云上传主要是用的plupload上传机制下面的当我没说只是有这个理念没试验过，我现在没空
参考文献 vue+element+plupload上传图片组件vue-plupload-aj 在vue中使用plupload上传图片到七牛(着重解决moxie is not defined问题) 前端上传组件Plupload使用指南前端上传组件Plupload使用指南
反正我现在不想弄了如果想弄自己看看吧，结果发现贼麻烦没遇到需求的时候我不会弄的
